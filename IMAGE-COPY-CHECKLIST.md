## File Naming Rules

✅ **DO:**
- Use exact filenames from this checklist
- Use lowercase extensions: `.jpg` or `.jpeg`
- Keep hyphens in filenames: `villa-street-view-01.jpg`

❌ **DON'T:**
- Add spaces: ~~`villa street view 01.jpg`~~
- Use uppercase: ~~`VILLA-STREET-VIEW-01.JPG`~~
- Change the name: ~~`villa-street-1.jpg`~~
- Use different extensions: ~~`.png`~~ (unless original is PNG)
