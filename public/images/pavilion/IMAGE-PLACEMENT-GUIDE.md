# Image Placement Guide for The Pavilion

This guide explains where to place the 27 images you've provided for The Pavilion website update.

## Directory Structure

All images should be placed in: `public/images/pavilion/`

Subdirectories:
- `exteriors/` - Villa exterior and community images
- `interiors/` - Interior living spaces, kitchens, bedrooms
- `recreation-zone/` - Recreation zone aerial and facility images
- `entrance/` - Main entrance gate images
